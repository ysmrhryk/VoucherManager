# 概要
- まだ未完成のため、今後開発を進める予定です。
- タイトル：伝票・請求書管理システム
- コンセプト：手書きやエクセルで作成される紙書類を減らしたい

# 主な機能

- 伝票管理：売上、見積書、請求書、入出金を管理
- 請求集計：月次請求書を一括生成
- PDF発行：請求書や売上伝票などの各種帳票をTCPDFで出力
- ログ管理：ユーザー別の操作履歴を記録
- 設定管理：自社情報（社名、振込先など）、支払方法、担当者ユーザーの複数管理など

# 技術スタックと選定理由

開発元が大手かどうか、採用例が多いかどうかなど、長期的に安定して使用できるかどうかを一番気にしています。

## バックエンド

### Laravel 11

- 圧倒的なシェアと熟成したエコシステム
- セキュリティと開発速度を両立できる

### Spatie Laravel Query Builder

- 検索条件に伴う複雑な条件分岐を排除

### TCPDF

- ユーザーの環境に依存せず、常に同じレイアウトのPDFを生成できる

## フロントエンド

### Vue 3

- Vue RouterやPiniaと併せて使用
- React等、他のフレームワークと比較して学習コストが低いと感じます。
- Laravelとの親和性も高く、連携がスムーズ。

### Bootstrap

- コンポーネントが豊富。デザインの一貫性と開発スピードに優れる。
